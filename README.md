# NodePay 账号管理工具

这是一个用于 NodePay 平台的账号管理工具集，包含注册和登录功能。

## 功能特点

### 注册工具 (main.py)
- 支持批量注册账号
- 支持使用真实邮箱列表或随机生成邮箱
- 支持多个验证码服务平台
- 支持代理IP功能
- 自动保存账号信息

### 登录工具 (login.py)
- 批量登录已注册账号
- 自动更新 Token
- 支持验证码自动处理
- 详细的登录状态日志


安装依赖：

pip install -r requirements.txt
